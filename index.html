UX Blueprint Specs

AeroLux Mobility

Urban eVTOL Flight Solutions

Homepage Specifications

1

Consistent Navigation Bar

Features a simplified menu structure with a high-contrast action CTA ("Book Flight"). Navigational elements focus on user exploration pathways.

Strategic Intent: Friction Reduction
2

Value Proposition & Visual Hero

Headline emphasizes emotional connection ("Elevate Your Daily Commute"). A large visual image frame placeholder ensures immediate engagement.

Strategic Intent: User Trust
3

Dual Action CTAs

Primary CTA "Reserve Now" leads directly to flight booking wizard; Secondary CTA "Explore Fleet" routes users to educate themselves on the aircraft.

Strategic Intent: Funnel Routing
4

3-Column Feature Grid

Details the value drivers: Zero Emissions, Time Efficiency, Luxury Comfort. Uses icon boxes to create digestible chunks of information.

Strategic Intent: Value Communication
5

Interactive Route Map

Allows users to input destination and origin to discover route networks, making flight options feel concrete and immediate.

Strategic Intent: Interactive Engagement
6

Conversion-Optimized Footer

Structured columns contain standard navigation, legal, and a clear newsletter subscription CTA to capture secondary traffic leads.

Strategic Intent: Lead Capture

Fleet Subpage Specifications

7

Navigation Breadcrumbs

Indicates clear parent-child structure ("Home > Fleet > Our Aircraft"), helping search engines index hierarchy and orienting the user.

Strategic Intent: SEO & Wayfinding
8

Filter & Sort Toolbar

Lets users filter eVTOL models by capacity, class, and range. Helps narrow decisions without overwhelming layout complexity.

Strategic Intent: Choice Simplification
9

Vibrant Product Showcase

Each eVTOL card displays aircraft class labels, beautiful image placements, and core descriptive summary copy to qualify product value.

Strategic Intent: Product Positioning
10

Core Specifications Checklist

Highlights capacity, range, and cruise speed in clear horizontal blocks. Allows easy cross-card evaluation scan.

Strategic Intent: Cognitive Scannability
11

Dynamic Product CTA Actions

"Configure & Book" directs users into customized color/interior select options, building high-intent user interest prior to reservation.

Strategic Intent: Personalization
12

Detailed Comparison Matrix

A tabular design comparing lower-level specifications (propulsion, noise dBA, luggage cap) to support analytical decision-making.

Strategic Intent: Trust & Transparency
